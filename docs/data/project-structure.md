# Project Structure and Naming

Main shared location:

```text
/project/holstegelab/Share/
```

Recommended structure:

```text
projects/
    PROJECT_NAME/
        raw_data/
        processed/
        results/
        logs/
references/
containers/
envs/
```

Naming recommendations:

- No spaces in file names
- Use lowercase and underscores
- Use dates as `YYYYMMDD`
- Avoid ambiguous names such as `final_v2_new_fixed`
