# Genomics of Neurodegenerative Diseases and Aging Lab

This documentation covers lab infrastructure usage, dry-lab workflows, and data management practices.

## Main sections

- **Start here**: first login, access model, and quick safety rules.
- **Dry lab**: Linux basics, Slurm on Spider, environments, containers, tmux, and Git workflow.
- **Data management**: project structure, storage and transfer, GDPR constraints, and good practices.
- **Wet lab**: reserved for wet-lab SOPs and sample handling workflows.

## Scope

This site focuses on infrastructure and operational guidance.
Genomics analysis pipelines are documented separately.
